## 🔥 Roulette
Roulette game for fun 

## ❤️ Install 
```
git clone https://github.com/Lluciocc/Roulette.git
```

After clone the project:

LINUX
```
sudo python3 requirements.py
```

WINDOWS
```
python requirements.py
```

## 🎮 PLAY 
LINUX
```
sudo python3 index.py
```

WINDOWS
```
python index.py
```
